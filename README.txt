1. Use Visual Studio Code

2. Make sure to use Google Chrome Canary
   https://www.google.com/chrome/canary/

3. Launch a local server 

   How to run a local server: (Part 1, 2)
   1. Type "npm install -g local-web-server" in terminal while being in project directory
   2. Type "ws" to launch a local server
   3. Navigate to "http://localhost:8000" in Google Chrome Canary

   How to run a local server: (Part 3)
   1. Type "npm install" in terminal while being in project directory
   2. Type "ws" to launch a local server
   3. Navigate to "http://localhost:8000/dist/" in Google Chrome Canary

4. Make sure to enable SharedArrayBuffer

   Windows:
   1. Rigth click Google Chrome Canary
   2. Click Properties
   3. In Target: Add "--enable-features=SharedArrayBuffer" at the end
