1. Make sure to use Google Chrome Canary
   https://www.google.com/chrome/canary/

2. Make sure to launch a local server

   How to run a local server:
   1. Type "npm install -g local-web-server" in terminal while being in project directory
   2. Type "ws"
   3. Navigate to "http://localhost:8000" in Google Chrome Canary

3. Make sure to enable SharedArrayBuffer

   Windows:
   1. Rigth click Google Chrome Canary
   2. Click Properties
   3. In Target: Add "--enable-features=SharedArrayBuffer" at the end
